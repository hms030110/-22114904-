# 자바 기말 프로젝트: 시간표 구현
#1. 프로젝트 주제 및 선정 배경
* 주제: 시간표 관리와 시간표를 보며 정리하기좋은 시간표 안에 메모장 밑 여러 기능
* 선정 배경: 내가 자주 사용하는 것을 구현할려고 찾아보니 대학생들이 자주 보는 시간표와 필기노트 같은 메모장을 한번에 구현하면 좋겠다라는 생각이 들어 구현해보았습니다.

#2. 주요 기능
* 시작/종료 시간 설정을 통한 유동적인 시간표 그리드 생성
* 요일별, 시간별 일정 추가 및 툴팁을 활용한 다이얼로그 메모장 기능
* `Serializable`을 활용한 시간표 데이터 로컬 파일(.dat) 저장 및 불러오기
* `Timer` 스레드를 활용한 실시간 시계 및 돌림판 회전 애니메이션 구현

#3.코드 작성 및 참조 출처 명시 (매우 중요)
